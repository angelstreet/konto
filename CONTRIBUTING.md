# Contributing

Thanks for your interest in Konto!

## Bug Reports & Feature Requests

Open an issue. Include screenshots if it's a UI bug.

## Code Contributions

Please open an issue first to discuss what you'd like to change.
Unsolicited PRs may be closed without review.

## Rules

- No hardcoded secrets or personal data
- `tsc --noEmit` must pass
- Test locally before submitting
- One PR per feature/fix

## Not Accepting PRs For

- Major architecture changes
- New integrations (bank providers, etc.)
- Translations (yet)
